Multiple Question Quiz (with random questions, 10 second timer and 3 lives)

A quiz with multiple questions from this GQuiz in the CMS. The GQuiz is generated via a plug-in using the following call which is assigned to the 'Start Quiz' button. In the gquizAd call several variables can be set, like whether a timer is needed, number of lives (if any), if correct answers are multipled by a set number and if the questions are shown in a random order.

The latest entries can also be requested by giving a classname of an element that should show them.

$('#gquiz1').gquizAd({
	'campaign_id' : '1',
	'entries_id' : '.entries',
	'entries_count' : 20
});

$('#gquiz1 #start_quiz').on('click', function(e) {
	e.preventDefault();
	$('#gquiz1').gquizAd({
    	'campaign_id' : '1',
    	'container_id' : '.quiz-questions',
    	'randomise' : true,
    	'score_multipler' : 10,
    	'timer' : 10,
    	'lives' : 3
    });
});

Single Question Quiz

We can also handle a quiz with a single question. The question is displayed on page load, without a 'Start Quiz' button. In the CMS, correct answers simple get a score of 1. Incorrect answers will just be 0.

$('#gquiz3').gquizAd({
	'campaign_id' : '14',
	'quiz_type' : 'single',
	'container_id' : '.quiz-questions'
});
