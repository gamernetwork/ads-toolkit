GPoll Demo Layouts

Poll Answers

List of answers from the GPoll database, in the order set in the CMS. Each answer is a checkbox. Images and extra content can be set locally. The GPoll is generated via a plug-in using the following call...

$('#gpoll1').gpollAd({
	'campaign_id' : '13',
	'question_id' : '55',
	'answers_id' : '.vote-answers',
	'response_count' : '3',
	'graph_id' : '.graph-answers',
	'extras' : answers_extras
});

Poll Graph

Poll responses graphed on the x-axis. Width is 100%, height is fixed. The most popular result is given the class .top.

Poll Graph (Y-Axis)

This is a second instance of GPoll, this time graphing the results on the y-axis. The GPoll call only asks for a graph on the y-axis...

$('#gpoll2').gpollAd({
	'campaign_id' : '13',
	'question_id' : '55',
	'answers_id' : '',
	'response_count' : '3',
	'graph_id' : '.graph-answers',
	'graph_axis' : 'y',
	'extras' : answers_extras
});
